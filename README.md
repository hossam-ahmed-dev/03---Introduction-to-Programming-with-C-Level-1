# C++ Programming Fundamentals and Basics

Welcome to my C++ learning journey repository! This repository contains a collection of fundamental C++ programs designed to practice core programming logic, foundational syntax, data types, and problem-solving exercises. 

## 🚀 Repository Contents

The code covers the following C++ foundational topics:
* **Basic Syntax & Output:** Printing text, handling escape sequences (`\n`, `\t`, `\"`, `\\`), and learning the difference between `\n` and `std::endl` (buffer flushing).
* **Data Types & Sizes:** Exploring fundamental types (`int`, `float`, `double`, `char`, `string`, `bool`) and determining their byte sizes/ranges using `sizeof()`.
* **Variables & Constants:** Practice with descriptive naming conventions, variable declaration, swapping values, and working with `const` constants like `PI`.
* **Interactive I/O:** Reading user input via `cin` and outputting structured text using `cout`.
* **Operators & Expressions:** Implementation of arithmetic, relational (`==`, `!=`, `<`, `>`), and logical operators (`&&`, `||`, `!`), alongside post/pre-increment behaviors (`A++` vs `++A`).
* **Mathematical Operations (`<cmath>`):** Utilizing library functions such as `sqrt()`, `pow()`, `round()`, `ceil()`, `floor()`, and `abs()`.

## 📈 Included Practice Projects / Exercises
Within the code snippets, you'll find custom programs to solve real-world calculation problems:
1. **Geometric Calculators:** Area computations for rectangles, triangles, circles (given diameter, radius, inscribed in squares, or isosceles triangles).
2. **Financial Math:** Total bill calculators including taxes/tips, parsing cash change remainders, and calculating monthly loan payments or settlements.
3. **Time & Currency Converters:** * Converting days, hours, minutes, and seconds into total seconds.
   * Parsing a massive duration of seconds back into structural `Days:Hours:Minutes:Seconds`.
   * Breaking down pocket coins (pennies, nickels, dimes, quarters, dollars) into absolute totals.
4. **General Logic:** Grade average calculators, future age calculators, simple value-swapping mechanisms, and string-reversing outputs.
